# 상품등록 서포터 (19기용)

올웨이즈/토스쇼핑 상품 등록을 도와주는 AI 에이전트입니다.
상품명, 가격, 옵션명, 태그, 상세페이지 기획을 한 번에 생성해줍니다.

---

## 설치 방법 (3단계)

### 1단계: 압축 해제
다운받은 zip 파일의 압축을 풀어주세요.

### 2단계: Claude Code에서 폴더 열기
터미널을 열고 다음 명령어를 입력하세요:
```bash
cd 상품등록-서포터
claude
```

### 3단계: 바로 사용!
아래 사용법을 참고해서 자유롭게 사용하세요.

---

## 사용법

### 방법 1: 자연어로 말하기 (가장 쉬움)
```
사과 상품 등록하고 싶어
```
```
상호명 황금상회, 사과 5kg, 원가 8000원으로 상품 만들어줘
```
부족한 정보가 있으면 하나씩 물어봅니다.

### 방법 2: 슬래시 커맨드
```
/상품등록
```

### 방법 3: 엑셀 파일로 한꺼번에
```
이 엑셀 파일로 상품 등록해줘 /Users/내컴퓨터/상품목록.xlsx
```

### 방법 4: 도매사이트에서 상품 찾기
```
도매사이트에서 사과 찾아줘
```
브라우저가 자동으로 도매사이트에 접속합니다. 로그인만 직접 해주세요.

---

## 생성되는 결과물

| 항목 | 설명 |
|------|------|
| 올웨이즈 상품명 | 3개 안 (★별표 + 키워드 중복 방식) |
| 토스쇼핑 상품명 | 3개 안 (깔끔한 설명형) |
| 가격표 | 공격적/기본/널널 3단계 + 플랫폼별 세팅값 |
| 옵션명 | 황금코볼트식 (수식어 포함) |
| 태그 | 5~6개 (올웨이즈용) |
| 상세페이지 기획 | 간이/보통/집중 3단계 선택 |

---

## 설정 변경

기본 설정을 바꾸고 싶으면 대화 중에 말하면 됩니다:

| 변경 사항 | 말하는 방법 |
|----------|-----------|
| 마진 공격적 (x1.5~1.6) | "마진을 공격적으로 해줘" |
| 마진 널널 (x2.2) | "마진 널널하게 해줘" |
| 옵션명 베이직 | "옵션명 베이직으로 해줘" |
| 상세페이지 간이 | "상세페이지 간이로 해줘" |
| 상세페이지 집중 | "상세페이지 집중으로 해줘" |
| 올웨이즈만 | "올웨이즈만 해줘" |
| 토스만 | "토스만 해줘" |
