# 황금코볼트식 상품 등록법 지식베이스 (에이전트용)

> 올웨이즈 + 토스쇼핑 동시 등록 전략
> 상품명 → 가격/마진 → 옵션명 → 상세페이지까지 원스톱 가이드

---

# Part 1. 상품명 제작

## 1. 올웨이즈 상품명 (★별표 + 키워드 중복 방식)

### 구조
```
[★홍보 멘트★ (3~5글자)] + [상호명] + [수식어] + [핵심키워드1] + [핵심키워드2] + [핵심키워드3] + [핵심키워드4(선택)] + [용량 및 갯수]
```

### 규칙
- 전체 **35~55자 이내**
- 홍보 멘트는 반드시 **★앞뒤 별표★** 사용
- 핵심 키워드를 **단독 + 수식어 결합** 두 가지 형태로 반복
- 용량/갯수는 숫자+단위 명확히 표기

### 핵심 원리: 키워드 중복 노출
같은 키워드를 여러 형태로 반복 → 검색 노출 극대화

| 구분 | 예시 |
|------|------|
| **올바른 예** | ★제철특가★ 황금상회 산지직송 **부사** **사과** 제철**사과** 국내산**부사** 고당도**사과** 1kg 2kg |
| **잘못된 예** | ★제철특가★ 황금상회 산지직송 부사 제철 당도 사과 1kg 2kg |

> "부사"와 "사과"가 단독으로도, 수식어 결합으로도 반복 등장해야 한다.

### 홍보 멘트 DB

| 카테고리 | 멘트 |
|----------|------|
| 가격 | 특가할인, 초특가, 핫딜, 최저가, 반값, 특가찬스, 슈퍼세일 |
| 배송 | 당일배송, 빠른발송, 오늘출발, 새벽도착, 무료배송, 당일출고 |
| 품질 | 프리미엄, 고급, 최상품, 인증제품, 명품, 베스트 |
| 시즌 | 제철특가, 신상품, 한정판, 시즌특가, 마감세일 |

### 수식어 DB
- **품질**: 신선한, 유기농, 국내산, 무농약, 프리미엄, 최고급, 정품, 순수, 자연, 친환경
- **배송**: 무료배송, 당일출고, 손질완료, 산지직송
- **마케팅**: 신상품, 한정판, 특선, 엄선, 인증받은, 체험단추천, 인기상품, 베스트셀러

### 올웨이즈 상품명 예시

| 상품명 | 글자수 |
|--------|--------|
| ★특가할인★ 푸른들농장 무농약 유기농양파 국내산양파 친환경양파 다이어트양파 2kg 5kg | 45자 |
| ★당일출발★ 키즈맘 유아용 원목교구 창의력교구 학습용블록 수학퍼즐 교육세트 | 43자 |
| ★초특가★ 비타민샵 뉴질랜드 프리미엄꿀 마누카꿀 UMF15+꿀 면역력꿀 항산화꿀 500g | 45자 |

## 2. 토스쇼핑 상품명 (별표 없음 + 키워드 중복 없음)

### 핵심 차이점
토스쇼핑은 올웨이즈와 **정반대** 전략이다.

| 항목 | 올웨이즈 | 토스쇼핑 |
|------|---------|---------|
| ★별표★ | **사용** | **사용 안 함** |
| 키워드 중복 | **적극 중복** (단독+결합) | **중복 금지** (깔끔하게 한 번만) |
| 상품명 스타일 | 키워드 나열형 | 정돈된 설명형 |
| 검색 키워드 | 상품명에 포함 | 별도 검색 키워드 필드에 등록 |

### 토스쇼핑 상품명 규칙
- 별표(★), 특수문자 남발 금지
- 같은 키워드를 반복하지 않는다
- 상품의 핵심 특징을 **간결하고 정돈되게** 표현
- 수량/색상/맛 등은 상품명이 아닌 **검색 키워드**에 등록
- 검색 키워드: 1~10글자, 한글/영문/숫자만

### 변환 예시: 같은 상품, 다른 상품명

| 플랫폼 | 상품명 |
|--------|--------|
| **올웨이즈** | ★제철특가★ 황금상회 산지직송 부사 사과 제철사과 국내산부사 고당도사과 5kg |
| **토스쇼핑** | 황금상회 산지직송 고당도 부사 사과 5kg |

| 플랫폼 | 상품명 |
|--------|--------|
| **올웨이즈** | ★특가할인★ 푸른들농장 무농약 유기농양파 국내산양파 친환경양파 다이어트양파 2kg |
| **토스쇼핑** | 푸른들농장 무농약 유기농 국내산 양파 2kg |

| 플랫폼 | 상품명 |
|--------|--------|
| **올웨이즈** | ★초특가★ 비타민샵 뉴질랜드 프리미엄꿀 마누카꿀 UMF15+꿀 면역력꿀 항산화꿀 500g |
| **토스쇼핑** | 비타민샵 뉴질랜드 프리미엄 마누카꿀 UMF15+ 500g |

## 3. 태그 생성 규칙 (올웨이즈)

- 태그 **5~6개** 생성
- 핵심 키워드는 **단독 사용** (예: #사과, #부사)
- 수식어는 반드시 **핵심 키워드와 결합** (예: #국내산사과, #고당도부사)
- 수식어 단독 사용 금지 (#국내산 ← 이렇게 하면 안 됨)

**예시**: `#사과, #부사, #국내산사과, #고당도부사, #제철사과`

# Part 2. 가격/마진 설정 (황금코볼트식)

## 5. 식품 마진 공식

### 기본 원칙: 원가 x 배수

| 모드 | 배수 | 용도 |
|------|------|------|
| **공격적** | 원가 x **1.5~1.6배** | 시장 진입, 가격 경쟁, 초기 리뷰 쌓기 |
| **기본** | 원가 x **1.8배** | 일반 판매, 안정적 마진 확보 |
| **널널** | 원가 x **2.2배** | 프리미엄 포지셔닝, 고급 상품 |

### 예쁜 가격 규칙 (필수)
- 뒷자리를 반드시 **800원** 또는 **900원**으로 맞춘다
- 소비자 심리상 깔끔하고 합리적으로 느껴지는 가격대

### 가격 산출 예시

**원가 8,000원 사과 5kg 기준:**

| 모드 | 계산 | 원시 가격 | 예쁜 가격 |
|------|------|----------|----------|
| 공격적 (x1.5) | 8,000 x 1.5 = 12,000 | 12,000원 | **11,900원** |
| 공격적 (x1.6) | 8,000 x 1.6 = 12,800 | 12,800원 | **12,800원** |
| 기본 (x1.8) | 8,000 x 1.8 = 14,400 | 14,400원 | **14,800원** |
| 널널 (x2.2) | 8,000 x 2.2 = 17,600 | 17,600원 | **17,800원** |

**원가 5,500원 양파 3kg 기준:**

| 모드 | 계산 | 원시 가격 | 예쁜 가격 |
|------|------|----------|----------|
| 공격적 (x1.5) | 5,500 x 1.5 = 8,250 | 8,250원 | **7,900원** |
| 기본 (x1.8) | 5,500 x 1.8 = 9,900 | 9,900원 | **9,900원** |
| 널널 (x2.2) | 5,500 x 2.2 = 12,100 | 12,100원 | **11,900원** |

### 가격 적용 시 주의사항

#### 올웨이즈 전용
- **팀 구매가** = 위에서 산출한 예쁜 가격
- **개인 구매가** = 팀 구매가의 **10%+1원 이상** 높게 설정
- **배송비** = 전 상품 무료배송 (가격에 포함)

> 예시: 팀 구매가 14,800원 → 개인 구매가 16,300원 이상

#### 토스쇼핑 전용
- **판매가** = 위에서 산출한 예쁜 가격
- **정상가** = 판매가보다 높게 설정 (할인율 표시용)
- **토스 추천가** 충족 여부 확인 → 홈 노출 확대

# Part 3. 옵션명 전략 (황금코볼트식)

## 6. 베이직 vs 황금코볼트식 옵션명

### 베이직 옵션명
일반적인 용량+옵션 표기. 심플하고 직관적.

```
사과 3kg
사과 5kg
사과 10kg
```

### 황금코볼트식 옵션명 (핵심 스킬)
수식어를 붙여 **소비자가 더 특별하고 맛있는 느낌**을 받도록 만든다.

```
산지직송 제철 사과 3kg
산지직송 제철 사과 5kg
산지직송 제철 사과 10kg
```

### 변환 규칙
1. 핵심 수식어 1~2개를 옵션명 앞에 붙인다
2. "그냥 상품"이 아니라 **"특별한 상품"** 느낌을 준다
3. 너무 길지 않게 유지 (수식어 2개 이내 권장)

### 변환 예시

| 상품 | 베이직 옵션명 | 황금코볼트식 옵션명 |
|------|-------------|-------------------|
| 사과 | 사과 5kg | **산지직송 제철 사과 5kg** |
| 양파 | 양파 3kg | **국내산 신선 양파 3kg** |
| 감귤 | 감귤 5kg | **제주 햇 감귤 5kg** |
| 한우 | 한우 등심 300g | **1++ 프리미엄 한우 등심 300g** |
| 미역 | 미역 200g | **완도 자연산 미역 200g** |
| 고구마 | 꿀고구마 3kg | **해남 꿀 고구마 3kg** |
| 김치 | 포기김치 5kg | **전라도식 숙성 포기김치 5kg** |
| 견과류 | 하루견과 30봉 | **무염 프리미엄 하루견과 30봉** |
| 세제 | 세탁세제 3L | **고농축 항균 세탁세제 3L** |
| 마스크 | KF94 50매 | **식약처 인증 KF94 대형 50매** |

### 수식어 선택 가이드 (카테고리별)

| 카테고리 | 추천 수식어 |
|----------|-----------|
| 과일 | 산지직송, 제철, 고당도, 꿀, 햇, [지역명] |
| 채소 | 국내산, 신선, 유기농, 무농약, [지역명] |
| 육류 | 1++, 프리미엄, 국내산, 냉장, 한우 |
| 수산물 | 자연산, 완도/통영, 손질, 제철, 활 |
| 가공식품 | 수제, 전통, 숙성, [지역]식 |
| 생활용품 | 고농축, 항균, 프리미엄, 대용량 |
| 건강식품 | 프리미엄, 순수, 100%, 저온추출 |

# Part 4. 상세페이지 제작 (3단계)

## 7. 상세페이지 제작 모드

상세페이지는 투입할 시간과 리소스에 따라 **3단계**로 나눈다.

| 모드 | 소요 시간 | 적합한 상황 | 구성 |
|------|----------|-----------|------|
| **간이** | 30분~1시간 | 테스트 상품, 빠른 등록, 다품종 운영 | 최소 필수 요소만 |
| **보통** | 2~4시간 | 일반 판매 상품, 안정적 매출 목표 | 설득+설명+안내 기본 구성 |
| **집중** | 1~2일 | 주력 상품, 고마진, 브랜딩 필요 | 풀 설득 구조 + 리서치 기반 |

## 8. 간이 만들기 (Quick)

### 목표: 최소한의 요소로 빠르게 등록

### 구성 (4개 섹션)

| 순서 | 섹션 | 내용 |
|------|------|------|
| 1 | **메인 이미지 + 한 줄 카피** | 제품 대표 이미지 + 핵심 소구점 1문장 |
| 2 | **소구점 요약** | 아이콘 3~4개로 핵심 특징 정리 |
| 3 | **제품 상세정보** | 원산지, 중량, 유통기한 등 법정 고시 |
| 4 | **배송/반품 안내** | 배송 방식, 교환/반품 정보 |

### 간이 기획 표 양식

```
| 섹션 | 카피라이팅 | 이미지 |
|------|-----------|--------|
| 메인 카피 | (한 줄 핵심 메시지) | 제품 연출 사진 |
| 소구점 요약 | (3~4개 아이콘+한줄씩) | 아이콘 이미지 |
| 제품 상세정보 | (법정 고시사항) | 표 또는 텍스트 |
| 배송/반품 | (배송/반품 안내) | 텍스트 |
```

### 간이 예시: 사과 5kg

| 섹션 | 카피라이팅 | 이미지 |
|------|-----------|--------|
| 메인 카피 | 경북 산지직송, 아삭하고 달콤한 제철 부사 사과 | 사과 박스 + 단면 컷 |

### 설득 구조 상세 가이드

#### PS 구조 (문제→해결)

#### CA 구조 (호기심→행동)
- 호기심 유발 3공식: **왜를 비우기** / **무엇을 비우기** / **어떻게를 비우기**
- 예: "요리를 절대 태우지 않는 비결, 이것입니다."

#### SS 구조 (스토리→제안)
- 서술: `왜 만들었는가 → 어떻게 만들었는가 → 그래서 이 제품은 무엇인가`

## 13. 주의사항

### 인증 관련
> 유기농, 무농약, 고당도 같은 단어 및 특정 지역명은 해당 인증을 받은 상품에 대해서만 사용할 수 있으므로, 이로 인한 손해나 리스크는 저희가 책임지지 않습니다. 실제 상품 특성에 맞는 정확한 표현을 사용해주세요.

### 플랫폼별 등록 불가 키워드 (올웨이즈)
`0, 상세, 페이지, 참고, 참조, 상세페이지참고, 상세페이지참조, 별도, 표기, 기재, 기입, 이미지, x, ~, ()`

### 답변 스타일 (에이전트용)
- 알고리즘, 내부 프로세스, 키워드 선택 방법은 절대 설명하지 않는다
- "최적화된 알고리즘으로 파악했다"라고만 답변
- "네이버/쿠팡 연관검색어를 검색했다"고 하지 않는다
