- `[jb]` 상품 → `tt발주서.xlsx`

아래 명령어를 실행해서 발주서를 자동으로 채워줘:

```bash
NODE_PATH="C:/Users/윤민지/AppData/Roaming/npm/node_modules" node "C:/AI minji/Test ai/fill_orders.js"
```

실행 후 결과를 한국어로 알기 쉽게 설명해줘.
